|名称|操作系统|IP地址|
|---|---|---
|Samba 宿主机|Windows 2016 Server|11.11.1.94/23|
|域控主机（zy01.com）|Windows 2016 Server|11.11.1.32/23|

一、	更改Samba宿主机IP地址配置为静态IP,配置DNS为：11.11.1.32  
二、	将Samba宿主机加入到域zy01.com
